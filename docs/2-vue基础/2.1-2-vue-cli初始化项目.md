
## vue-cli初始化项目(vue-cli搭建开发环境)

vue-cli@3
vue@2.5

npm install -g @vue/cli
or
yarn global add @vue/cli

### vue-cli 指定版本安装



3.0 以下😔

npm install -g vue-cli@版本号

3.0 以上😀

npm install -g @vue/cli@版本号

### 查看版本号

vue -V
or
vue --version

### 创建项目

vue create 1-vue-cli-start
